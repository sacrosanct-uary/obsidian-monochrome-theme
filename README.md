# Monochrome
## Custom icon
A matching custom icon for both the desktop and taskbar can be found in the GitHub repository:
- Download the `icon.ico` file from the [repository](https://github.com/sacrosanct-uary/obsidian-monochrome-theme) and move it to the folder for the theme.
#### To set the desktop icon:
- Navigate to the "Appearance" tab in Obsidian's settings and set a custom app icon, selecting the file.
#### To set the taskbar icon:
- Right-click on the Obsidian icon on your taskbar, right-click on the "Obsidian" label in the context menu, left-click on "Properties".
- Once inside the properties, select "Change icon", find and select the custom icon then restart your device for it to take effect.
